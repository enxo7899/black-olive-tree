# Black Olive Tree

A portfolio-grade, high-performance Mediterranean restaurant website showcasing award-winning UI/UX design.

## Features

- **Dual Character Theme**: Day/Night mode reflecting the venue's transformation from morning caf√© to evening cocktail bar
- **Smooth Scrolling**: Lenis-powered normalized momentum scrolling
- **Magnetic CTA**: Physics-based button tracking mouse movement
- **3D Elements**: React Three Fiber olive branch animation
- **Smart Menu**: Fuzzy search with Fuse.js and animated filtering
- **Performance**: Optimized for 100/100 Lighthouse scores

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion, Lenis
- **3D**: React Three Fiber, Three.js
- **Search**: Fuse.js
- **Icons**: Lucide React
- **TypeScript**: Strict mode

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
